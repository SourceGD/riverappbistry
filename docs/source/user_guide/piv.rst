.. _piv:

##########################################
PIV analysis
##########################################

Actually, there is no real screen for the PIV analysis, just a waiting screen.

If you are currently connected to internet,
the application will send the video and the project configuration on a server to process the PIV remotely.
This is done to free the user client machine from heavy calculations.

If you are not connected to internet, the PIV analysis will be done locally on your machine.
