====================================
RiverApp 2024 documentation
====================================

This documentation is divided into two main sections: the `User Guide` and the `Developer Guide`. The `User Guide` is intended for users who want to learn how to use RiverApp, while the `Developer Guide` is intended for developers who want to contribute to the development of RiverApp.
Since the application does not contain a mobile version yet, the documentation also contains an installation procedure.
