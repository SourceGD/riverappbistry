##################################################
How does the application work?
##################################################

Here is a diagram representing how the application works.
The vertically aligned boxes with big arrows represent the different screens of the application.

The little arrows starting from screens to other boxes represent the other files used by theses screens.
It can be utils, components, or just "back-end" files.
